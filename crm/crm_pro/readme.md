# 开发环境
* Python (3.7.4)
* Django (2.2.5)

# 程序的实现的功能

1. 开发通用的、可插拔式的增删改查组件，可很容易的嵌入各种django项目
2. 开发通用的、达到对象级别的权限控制组件，且允许用户自定制权限
3. 权限组件支持角色管理，根据业务需求分出销售、学员、讲师角色
4. 允许销售创建、跟进客户信息
5. 允许老师进行班级学生考勤管理、积分管理
6. 实现客户资源的公户与私户划分，销售可以对私户进行管理

# 程序的启动方式

python manage.py runserver 0.0.0.0:8000
